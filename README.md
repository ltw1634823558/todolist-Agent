# Todo List 应用

一个现代化的、响应式的 Todo List 前端应用，使用纯 HTML + CSS + JavaScript 开发。

## ✨ 功能特性

- ➕ 添加新任务
- ✅ 标记任务完成/未完成
- 🗑️ 删除任务
- 🔍 任务过滤（全部/待完成/已完成）
- 📊 实时统计（总任务/待完成/已完成）
- 💾 本地存储（刷新页面数据不丢失）
- 🎨 现代化渐变设计
- 🌙 支持深色模式
- 📱 完全响应式设计
- ⚡ 流畅的动画效果

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/ltw1634823558/todolist-Agent.git
```

2. 打开 `index.html` 文件即可使用

或者直接访问 [GitHub Pages](https://ltw1634823558.github.io/todolist-Agent/)

## 📖 使用说明

- 在输入框中输入任务内容，按 Enter 或点击 + 按钮添加任务
- 点击任务左侧的圆圈标记任务完成/未完成
- 点击任务右侧的垃圾桶图标删除任务
- 使用过滤按钮查看不同状态的任务
- 点击"清除已完成"按钮删除所有已完成的任务

## 🛠️ 技术栈

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins)
- LocalStorage API
